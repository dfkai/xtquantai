# 行为准则

本项目致力于为所有人提供友善、安全的参与环境。

## 我们的约定

- **友善与耐心**：社区里既有量化老手也有刚入门的朋友，回答问题时不嘲讽、不轻视。
- **就事论事**：讨论聚焦代码、策略逻辑与事实本身，不进行人身攻击。
- **尊重分歧**：量化方法没有唯一正确答案，欢迎摆数据、摆回测结果来讨论。
- **不发布违规内容**：不传播有版权的完整研报原文，不发布违反证券法规的内容，不进行荐股或承诺收益。

## 投资风险提示

本仓库的技能与脚本仅用于学习和研究，不构成任何投资建议。使用生成的策略进行实盘交易，盈亏自负。

## 执行

违反准则的行为（骚扰、刷广告、恶意攻击等）将视情节给予提醒、删除内容或封禁处理。如需举报，请发邮件至 davidfnck@gmail.com。
